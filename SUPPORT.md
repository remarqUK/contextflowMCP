# Support

## Where To Ask For Help

- Usage/setup questions: open a GitHub Discussion (or Issue if Discussions are not enabled).
- Bug reports: open a GitHub Issue using the bug template.
- Feature requests: open a GitHub Issue using the feature template.
- Security concerns: follow `SECURITY.md`.

## What To Include

- Your MCP client (`claude`, `codex`, `gemini` integration details)
- OS and Node.js version
- Server config (redact secrets)
- Exact tool/prompt call used
- Error message and expected behavior

## Non-Goals For Support

- Closed-source client internals outside this repository
- Environment-specific outages unrelated to this project
